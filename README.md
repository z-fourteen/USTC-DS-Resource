# Learning Resources for the Data Science (DS) Major at USTC

This repository collects **learning materials, notes, and practice resources** related to the **Data Science (DS) major at the University of Science and Technology of China (USTC)**. It is intended to serve as a structured reference for undergraduate students who want to build solid foundations, prepare for research, or improve employability.

The materials are organized according to core courses, supporting mathematics, programming skills, and advanced topics commonly encountered in the DS curriculum.
